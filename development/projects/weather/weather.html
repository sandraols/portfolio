How's the weather?

Vue.js & OpenWeatherMap API

A web app querying and displaying weather forecasts using OpenWeatherMap API, built with Vue.js. As a user types in a city the weather of that city is rendered. The app displays current weather and the forecast for the following five days. To add a visual experience the background color of the page changes depending on the current weather.
