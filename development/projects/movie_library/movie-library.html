Spectacular Movie Library

JavaScript, jQuery & OMDb (The Open Movie Database) API

A web app where the user can search for movies by it's title. If the movie exists in the OMDb API it will be displayed together with info about the movie such as poster, genre and plot. To make a nice user experience I also added several background themes the user can choose from.

Press any image to go to page.
