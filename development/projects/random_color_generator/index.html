Random Color Generator

javaScript project

This is a random color generator made with javaScript, HTML and CSS. Each time you click on one of the color boxes a random RGB color generates together with its value. I struggled a long time with the function of getting the correct rgb value match the colors displayed, but I finally managed to get it right. The page can also be viewed either in a light or a dark background of your choice.

Press any image to try it out!
