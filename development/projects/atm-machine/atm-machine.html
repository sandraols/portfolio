ATM Machine

JavaScript & localStorage

A digital ATM Machine where the user can log in to their bank account, check balance, withdraw and deposit money. There is also a feature that allows the user to create an account. Accounts and balance are stored with localStorage. The page is built with html, css and javaScript. The design is inspired by casino slot machines to make the user experience a bit more fun.
